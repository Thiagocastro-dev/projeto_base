# Projeto Base

Pequeno site demonstrativo com melhorias aplicadas: SEO, acessibilidade, responsividade, performance e PWA.

Como testar localmente:

1. Abra `src/index.improved.html` no navegador (ou copie o conteúdo para `src/index.html`).
2. Para testar o Service Worker em ambiente local, sirva a pasta com um servidor HTTP (por exemplo, `npx http-server .`).
